# facial-emotion-recognition
AI-based facial emotion recognition system developed using Python, OpenCV, TensorFlow, and CNN for real-time emotion detection.
# Models

This folder contains the trained deep learning model used in the AI-Based Facial Emotion Recognition System.

## Model Information

* Model Type: Convolutional Neural Network (CNN)
* Framework: TensorFlow / Keras
* Purpose: Facial Emotion Recognition

## Supported Emotions

* Happy
* Sad
* Angry
* Neutral
* Surprise

The model is used by the application to predict facial emotions from images and webcam input in real time.
