# facial-emotion-recognition
AI-based facial emotion recognition system developed using Python, OpenCV, TensorFlow, and CNN for real-time emotion detection.
# Pages

This folder contains the main application pages used in the AI-Based Facial Emotion Recognition System.

## Files

### Home.py

Main page of the application.

### Admin.py

Admin panel for managing application features and information.

### Image_Predict.py

Performs emotion prediction on uploaded images.

### Webcam_Predict.py

Performs real-time facial emotion recognition using webcam input.

### model_info.py

Displays information about the trained machine learning model.

### Train_Model.py

Used for training the Convolutional Neural Network (CNN) model.

## Purpose

These pages work together to provide image-based and real-time facial emotion recognition using Artificial Intelligence, Deep Learning, and Computer Vision techniques.
